# Car Game

This game is a car game created using HTML, CSS and JavaScript. The player uses the arrow keys to control the car and press up to jump the obstacles. 

## How to Play

- Use arrow keys to control the car.
- Press up to jump over obstacles.
- Avoid the obstacles.

## Instructions

1. Open the game and test via the link in the portfolio without installing or downloading any files.
2. Press the direction keys to move the car and press up to jump the obstacles.
3. Have a good time!
